# 🛠️ TapFunApp Support & Feedback Center

Welcome to the official support forum for **TapFunApp**. This is a centralized space to report bugs, suggest features, and discuss all our macOS/iOS products. 

> 💡 **Tip:** Before opening a new topic, please use the search bar at the top of the **Discussions** tab to see if your question has already been answered.

---

## 🚀 How to Get Help

We use **GitHub Discussions** to track everything. Please click the **[Discussions](https://github.com/tapfunapp/support/discussions)** tab above and choose the right category for your product:

* **📢 Announcements** — Official updates, release notes, and news from TapFunApp (Read-only).
* **🟢 FeedGarden** — RSS reader support, feed issues, and feature requests.
* **🖼️ FloatPic** — Desktop image viewer feedback, OCR, or EXIF issues.
* **⬇️ DropToDo** — Menu-bar timer configuration and gesture feedback.
* **✨ VibeCursor** — Cursor effects, particle trails, and audio assistance.
* **🎨 PixelColor** — Color picker, palette management, and hex format suggestions.
* **🤖 CodeRouter** — Claude Code API switching, keys, and model configuration.
* **📸 Snapi** — Lightweight image editor tool, cropping, and export options.

---

## 🐛 Reporting a Bug or Suggesting a Feature

When posting a new discussion, please try to include:
1. **Device Info:** Your macOS / iOS version.
2. **App Version:** Which version of the App you are using.
3. **Steps to Reproduce:** (For bugs) Clear steps or a screenshot/GIF showing the issue.

Thank you for supporting indie development! Every piece of feedback helps us make these tools better. ❤️
